## Featured Project: Rental Law Multi-Agent Resolution System

A **production-grade, security-hardened multi-agent system** designed for legal aid providers handling high-volume tenant rental disputes.

This system coordinates parsing, retrieval, validation, and grounded response generation while enforcing strict safety and auditability boundaries. Built with the same principles of state isolation, guardrails, deterministic orchestration, and defence-in-depth highlighted across my portfolio.

### What it does
- Ingests and grounds answers in official documents (`tenants_rights.pdf` + historical case archive `previous_judgements.pdf`)
- Uses hybrid retrieval (vector + keyword) + chunk validation before any LLM call
- Classifies every interaction with clear **exit reasons**: guardrail block, out-of-scope, HITL escalation, insufficient information, or successful resolution
- Maintains typed state isolation so each agent only sees the minimal data it needs
- Provides full LangSmith tracing + custom efficiency dashboard (resolution rate, latency, token usage, cost, tool calls)

### Production-Grade Design Principles
1. **Strict state isolation** — Limits blast radius of any malfunction or compromise
2. **All LLM inputs treated as untrusted** — Guardrail scanning on user queries, retrieved chunks, and clarification responses
3. **Deterministic orchestration** — Rule-based Python routing (not model-generated) for reliability and full auditability
4. **Defence-in-depth** — Security layered at system boundaries without slowing down core legal reasoning

**Tech Stack**: Python • LangChain • LangGraph • ChromaDB • OpenAI • Pydantic • pypdf • LangSmith

**Evaluation Highlights** (from live traces):
- Clear separation of successful resolutions vs. guardrail / out-of-scope / escalation cases
- Full cost, latency, and token telemetry
- Designed for regulated/sensitive domains where hallucination or prompt injection cannot be tolerated

This project showcases exactly the kind of **safe, auditable, business-aligned agentic system** I build.
